# vue3 vite 脚手架

## 安装

```bash
npm i @feiyuhao/vite_cli -g
```

## 项目设置

```bash
npm install
```

### 开发环境

```bash
node ./index.js create <app-name>
```

### 映射全局

```bash
npm run link
```

### 卸载全局

```bash
npm uni @feiyuhao/vite_cli -g
```

## 使用

|说明|命令|
|-|-|
|查看版本号|vc -V|
|查看信息|vc -h|
|创建项目|vc create <app-name>|
